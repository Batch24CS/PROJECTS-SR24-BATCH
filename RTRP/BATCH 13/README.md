# 🌍 Civic Issues Reporting App

A modern web-based platform that enables citizens to report civic problems directly to municipal authorities for faster resolution and better public service management.
The system improves communication between citizens and government departments by providing an easy way to register complaints, monitor progress, and manage civic issues efficiently.

The application helps in reporting issues such as:

* 🕳️ Road damage and potholes
* 💡 Street light failures
* 🗑️ Garbage dumping and sanitation issues
* 🚰 Water leakage problems
* 🚦 Traffic and public safety concerns
* 🌧️ Drainage and waterlogging complaints

# ✨ Features

## 👤 User Features

* User Registration & Login
* Report civic issues with descriptions
* Upload issue-related images
* Share current/live location
* Select issue categories
* View submitted complaints
* Track complaint status in real time
* Feedback submission system

## 🛠️ Admin Features

* Admin dashboard for complaint management
* View all complaints in one place
* Update issue status
* Manage and monitor reports
* Track resolved and pending issues
* Improve response efficiency

## 📍 Location Tracking

* Detects user location for accurate issue reporting
* Helps authorities identify issue areas quickly

## 📊 Complaint Status Tracking

Users can track issue progress through stages such as:

* Submitted
* Under Review
* In Progress
* Resolved

## 🔔 Real-Time Updates

* Faster communication between citizens and authorities
* Instant complaint status updates

# 🔧 Installation & Setup

Follow these steps to run the project locally.

## 1️⃣ Clone the Repository

git clone https://github.com/your-username/Civic-Issues-Reporting.git


## 2️⃣ Navigate to the Project Directory

cd Civic-Issues-Reporting

## 3️⃣ Install Required Packages

### Install Python Dependencies

pip install -r requirements.txt

### Install Node Modules

npm install


# ▶️ Run the Project

Start the application using:

python app.py

The server will start running on:


http://localhost:5000


# 🗄️ Database Configuration

Configure your database connection settings inside your configuration or `.env` file.

Example:

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=civic_issues
